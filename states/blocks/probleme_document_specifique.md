<p><b>Concernant votre document</b></p>

<p>Pour être accepté par la CMA, votre document doit respecter les critères suivants :</p>

<ul>
<li><b>Lisibilité</b> : Toutes les informations doivent être parfaitement lisibles</li>
<li><b>Intégralité</b> : Les quatre coins du document doivent être visibles</li>
<li><b>Actualité</b> : Le document doit être en cours de validité</li>
<li><b>Format</b> : JPG, PNG ou PDF, maximum 5 Mo</li>
</ul>

<p>Conseil : Prenez la photo dans un endroit bien éclairé, sur un fond contrasté (table sombre pour document blanc).</p>
