<p><b>Comment télécharger vos documents ?</b></p>

<ol>
<li>Connectez-vous sur <a href="https://www.exament3p.fr">ExamT3P</a> avec vos identifiants</li>
<li>Cliquez sur "Mes documents" dans le menu</li>
<li>Pour chaque document, cliquez sur "Parcourir" puis sélectionnez votre fichier</li>
<li>Cliquez sur "Envoyer" pour valider</li>
</ol>

<p><i>Formats acceptés : JPG, PNG ou PDF. Taille maximale : 5 Mo par fichier.</i></p>
